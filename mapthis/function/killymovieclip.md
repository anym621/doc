# killYMovieClip

`killYMovieClip(动画：YMovieClip，图层名：String)`



** 描述：**

 每次动画从舞台上清除时调用。



** 示例：**

```javascript
this.mapjs=this.MapNode.getComponent(this.MapStr);              
this.mapjs.killYMovieClip=function(ymc,LayerName){ 
     //要处理的内容                                                              
}
```
